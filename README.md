# NoteBookLM-CyberSecurity
Projeto com o objetivo de criar um assistente virtual voltado aos meus estudos de CyberSecurity, utilizando o modelo NotebookLM.

Como citado anteriormente, escolhi o tema de CyberSecurity, principalmente os relacionados aos fundamentos já que eu estou iniciando minha jornada no tema, baixei alguns livros que citarei posteriormente assim como utilizei alguns canais no YouTube de pessoas que julgo competentes em ensinar o assunto.

Meu objetivo é através desse projeto potencializar meus estudos na área de modo a me tornar um grande profissional blue team futuramente.

As minhas fontes iniciais foram os livros: 

- Cibersegurança para Leigos | Joseph Steinberg
- Segurança em Redes: Princípios e Práticas | William Stallings
- Blue Team Field Manual (BTFM) Volume I (Revised): Core Defensive and Analysis Techniques |  Robert J. Andrews
- Análise Prática de Malware | Michael Sikorski, Andrew Honig
- Threat Modeling | Adam Shostack
- Cybersecurity Blue Team Toolkit | Nadean H Tanner
